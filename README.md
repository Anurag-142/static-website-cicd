# Static Website CI/CD

## Project Overview

This project demonstrates a complete CI/CD pipeline using Azure DevOps.

## Technologies

- HTML
- CSS
- JavaScript
- Git
- GitHub
- Azure Repos
- Azure DevOps Pipelines

## Pipeline

The pipeline performs the following:

1. Checkout source code
2. Validate project files
3. Publish build artifacts

## Future Enhancements

- HTML validation
- CSS linting
- JavaScript linting
- Continuous Deployment to Azures
- Add a staging environment for review deployments
- Implement accessibility testing in CI
- Use versioned artifact storage for rollback support
