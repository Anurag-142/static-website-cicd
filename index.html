Satic Website CI/CD

Learn how Azure DevOps automates build, test, and deployment for a static website.

About this project

This website is built as a simple static project and deployed using Azure DevOps pipeline automation.

Build Automation

The pipeline compiles and packages assets automatically on every commit to the main branch.

Validation Checks

Basic file validation helps ensure the website structure is correct before deployment.

Fast Deployment

Deployment uses Azure CLI to publish the built static content directly to Azure Storage.

How it works

The pipeline builds the site, validates the files, publishes artifacts, and deploys the static content to Azure Storage.
